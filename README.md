# PML - Programable Markup Language
> 저는 HTML로 코딩해요.
```xml
<test.pml>
	<import from="std.cli">
		print
	</import>
	<func name="main" return="true">
		<arg name="argv"/>
		<body>
			<print>Hello World!</print>
			<return>0</return>
		</body>
	</func>
</test.pml>
```
